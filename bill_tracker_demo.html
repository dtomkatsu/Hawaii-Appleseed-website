Usage Instructions

Quick Start

  1. Include the CSS and JS files in your Squarespace Code Injection (Footer):
    <link rel="stylesheet" href="bill_tracker_widget.css"> <script src="bill_tracker_widget.js"></script>
  2. Add the tracker HTML with data attributes to your card:
    <div class="tfc-bill-tracker" data-tracker-id="unique-id" data-hb="476,477" data-sb="123" data-year="2026"> </div>
  3. The widget auto-initializes on page load!

Configuration Options

Attribute Description Example
data-tracker-id Unique identifier for this tracker (required) freezing-act-46
data-hb House Bill number(s), comma-separated 476 or 475,476,477
data-sb Senate Bill number(s), comma-separated 123 or 123,124
data-year Legislative session year (default: 2026) 2026

JavaScript API

// Manual initialization with config BillTrackerWidget.init({ trackerId: 'my-tracker', element: document.querySelector('#my-element'), hbNumbers: ['476', '477'], sbNumbers: ['123'], year: '2026', refreshInterval: 3600000 // 1 hour }); // Refresh all trackers BillTrackerWidget.refreshAll(); // Refresh specific tracker BillTrackerWidget.refresh('freezing-act-46'); // Clear cached data BillTrackerWidget.clearCache(); // Update configuration BillTrackerWidget.configure({ refreshInterval: 1800000, // 30 minutes cacheDuration: 900000 // 15 minutes });

Features

  • ✔ Auto-refresh: Updates every hour by default
  • ✔ Local caching: Reduces API calls with 30-minute cache
  • ✔ Multiple bills: Track multiple HB/SB per policy area
  • ✔ Toggle view: Switch between HB and SB feeds
  • ✔ Status badges: Visual indicators for bill progress
  • ✔ History timeline: Expandable history of all updates
  • ✔ CORS proxy: Works cross-origin with proxy fallback
  • ✔ Retry logic: Automatic retry on failed requests
